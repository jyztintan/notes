# Arrays

## Getting Elements

Array indexing. O(1)

## Searching Elements

Best case, v found at first pos. O(1)

Worst case, v not found in list, required O(N) scan to determine.

## Inserting Elements

- Insert at i = 0, shift all N elements. O(N).
- Insert at i = N, no shifting of elements. O(1).

## Removing elements

- Remove at i = 0, shift all N elements. O(N).
- Remove at i = N-1, no shifting of elements. O(1).
